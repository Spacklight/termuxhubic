# WebXPDF

## Description
Converts HTML code or HTML files into clean, distraction-free PDF documents.

## Features
- ✅ Converts HTML files to PDF
- ✅ Converts raw HTML code to PDF
- ✅ Strips out scripts and styles for clean output
- ✅ Preserves headings, paragraphs, and lists
- ✅ Professional formatting with headers and page numbers

## How to Use in Termux

### Step 1: Install required packages (one-time setup)
```bash
pip install fpdf2 beautifulsoup4
```

### Step 2: Download the script
```bash
curl -O https://raw.githubusercontent.com/spacklight/termuxhub/main/softwares/webxpdf.py
```

### Step 3: Make it executable
```bash
chmod +x webxpdf.py
```

### Step 4: Convert HTML to PDF

**Option A: Convert an HTML file**
```bash
python webxpdf.py article.html output.pdf
```

**Option B: Convert HTML code directly**
```bash
python webxpdf.py "<h1>Hello World</h1><p>This is a test.</p>" test.pdf
```

### Step 5: Open the PDF
```bash
termux-open output.pdf
```

## Requirements
- Python 3.x
- fpdf2 (`pip install fpdf2`)
- beautifulsoup4 (`pip install beautifulsoup4`)

## Examples

### Example 1: Convert a webpage
```bash
# First save the webpage as HTML
curl https://example.com > page.html

# Then convert to PDF
python webxpdf.py page.html example.pdf
```

### Example 2: Convert HTML code
```bash
python webxpdf.py "<h1>My Document</h1><p>Content here</p>" doc.pdf
```

## Output
The script generates a clean PDF with:
- Title at the top
- Properly formatted headings (H1, H2, H3)
- Paragraphs with proper spacing
- Bullet points for lists
- Page numbers at the bottom
- "Generated by WebXPDF" header
